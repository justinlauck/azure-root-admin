# Lauckserv Azure Root Administration 

This repo contains the Lauckserv Azure Root Administration Zone.

## Manual Prerequisites
To utilize this repository, the following manual prerequisites must be in place:

 1. GitHub Repository
 2. Root Azure Management Group
 3. Root Azure Management Subscription
 4. Azure Service Principal
 5. GitHub Pipeline

## Operations
The following operations are supported in this administration zone:

 - Manage top-level management groups and subscriptions
 - Boostrap all operational administration areas
